Content

Sets: An unordered list of non-repeating elements/members
Subsets: Sets whose elements/members fall entirely within the range of another set. All sets are subsets of itself, but a Proper Subset only applies when set =/= subset. (note: every set is a subset of itself!)
Supersets: If set A is a subset of set B, then set B is considered a superset of set A. Therefore, all sets are supersets of itself, as well. Once again,a Proper Superset only applies when superset=/=set.
Note: if two sets are subsets AND supersets of each other, then the two sets must be equal.
Power Subsets: When counting the number of possible subsets, each element in the set has the option to be included in the subset or excluded from the subset. As such, the number of possible subsets within a set is always equal to 2n, where n is equal to the number of elements within the set. This is called the Power Set of any set X, which is denoted by the symbol P(X).
The Universal Set: The Universe refers to all of the elements that are considered for a particular operation between sets. For example, the universe of all integers would be the set Z.
Set Complement: The complement of a set refers to all elements within a defined range (default the universe) that is not included within set X. This is expressed through the notation U / X.
Set Difference: A set compliment can be performed between sets within the same universe. The set differnce of set A to set B would be notated as "B / A".
Set Union: The union of two sets refer to all elements within the universe that is included in any sets within the union. A common way to think of it is to say "For Set A and Set B within Universe Z, the union of Set A and B would include all elements that are in universe Z that is in Set A OR Set B."
Set Intersection: The intersection of sets refers to all elements within the universe that is included in all sets within the intersection. A common way to think of it is to say "For Set A and Set B within Universe Z, the intersection of Set A and B would include only elements that are in universe Z that is in BOTH Set A and Set B."
De Morgan's Laws for Sets: De Morgan's Laws for sets refes to a pair of concepts, known as the Theorem and the Corollary.
The Theorem states that for Set A and B within Universe Z, the Set Complement of the Union of the two sets is equivalent to the Set Intersection of the complements of the two sets.
The Corollary states that for Set A and Set B within Universe Z, the Complement of the Intersection of Set A and Set B is equivalent to the Union of the Complement of Set A and Set B.
Symmetric Difference: The Symmetric Difference of Set A and B in Universe Z refers to the elements that are ONLY in Set A OR Set B, and NOT BOTH. (Exactly one of Set A and B).

Ordered Pair: An Ordered Pair refers to two or more elements placed together in a specific order.
Cartesian Product: The Cartesian Product of two or more sets refers to all ordered pairs consisting of an element of the first set followed by an element of the next set. If all sets in a cartesian product are finite, the size of the cartesian product is equivalent to the size of all individual sets multiplied together.
Partitions: The Partition of a set refers to all Disjoint, Nonempty subsets of that set whose union is that set. Each subset in this partition is referred to as a part, block, or cell of the partition.
